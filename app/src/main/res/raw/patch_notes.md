### What's New
**Overlay Media Type Selector**
- New setting to choose overlay media type for game marquees
- Options: Marquees, 3D Boxes, Covers, Screenshots, Fanart, MixImages
- Game marquee lookup now respects your selected media type with fallback to marquees

**ES-DE Settings Enhancements**
- Added "Reset to System Wallpaper" button to quickly clear ES-DE display
- Path settings now show actual default paths (e.g., `/storage/emulated/0/ES-DE/downloaded_media`) instead of just "Not set"
